# 📚 AI Study Planner

An AI-powered study planning web app that helps students organize their learning schedule, set goals, and stay on track.

## 🚀 Features

- AI-generated personalized study plans
- Subject and topic management
- Progress tracking
- Schedule visualization

## 🛠️ Tech Stack

- JavaScript
- CSS
- Vite

## 📦 Setup & Installation

```bash
# Clone the repo
git clone https://github.com/NandhiniVR/AI-Study-Planner.git

# Navigate into the project folder
cd "AI Study Planner"

# Install dependencies
npm install

# Start the development server
npm run dev
```
